# LeaseMate

Smart Tenant Management System

## Overview
LeaseMate is a digital platform that helps landlords and tenants manage rental activities efficiently. It replaces traditional ledger books by tracking rent payments, electricity bills, tenant details, and maintenance issues.

## Features

### Owner Side
- Property Management
- Room Management
- Tenant Management
- Rent Tracking
- Electricity Bill Tracking
- Security Deposit Tracking
- Issue Management
- Owner Dashboard

### Tenant Side
- Personal Dashboard
- Rent Breakdown
- Electricity Usage Comparison
- Expense Charts
- Issue Reporting

### Shared Living Features
- Room-based Multiple Tenants
- Automatic Rent Splitting
- Electricity Bill Splitting
- Splitwise-style Expense Tracker
